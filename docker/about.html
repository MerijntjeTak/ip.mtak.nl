About ip.mtak.nl
================
ip.mtak.nl is a free service providing you with the IP address that was used to request the page. It's functionality is similar to the likes of whatismyip.com, but fast and ad-free.

IPv4 and IPv6 enabled
=====================
There are several ways to request your IP address:

  • http://ip.mtak.nl/ Automatically selects IPv4 or IPv6 based on your OS's preference (usually IPv6 if available)
  • http://ipv4.mtak.nl/ will return your IPv4 address only
  • http://ipv6.mtak.nl/ will return your IPv6 address only

Due to technical limitations it's not possible to return both your IPv4 and IPv6 address at the same time. If you want to know both addresses, issue two seperate requests.

API
===
There is also an API version available which only returns the IP address (no other HTML). When you request the page http://ip.mtak.nl/ using one of the following User-Agent headers you will automatically get the API version:

  • m/curl/i
  • m/wget/i
  • m/lynx/i
  • m/emacs/i
  • m/links/i

You can also explicitly request the API version by requesting http://ip.mtak.nl/api/

Script examples
===============

Bash/Bourne shell
-----------------
$ EXTERNAL_IP=`curl -s http://ip.mtak.nl`
$ echo $EXTERNAL_IP
203.0.113.1

Ruby
----
require 'net/http'
external_ip = Net::HTTP.get(URI('http://ip.mtak.nl/api/'))

Perl
----
use LWP::Simple;
my $external_ip = get 'http://ip.mtak.nl/api/';

C# (.NET >3.5)
--------------
WebClient ipMtakNlClient = new WebClient();
string externalIp = ipMtakNlClient.DownloadString("http://ip.mtak.nl/api/");
